## 6. Next.js & 서버리스 API

- Next.js 프로젝트 생성:  
  pnpm create next-app nextjs-basic --typescript --eslint --app --src-dir --import-alias "@/*" --no-tailwind --yes

- REST API 테스트 (curl):  
  curl -X POST http://localhost:3000/api/todos -H "Content-Type: application/json" -d '{"title": "새 할일"}'  
  curl -X PUT http://localhost:3000/api/todos/1 -H "Content-Type: application/json" -d '{"completed": true}'
